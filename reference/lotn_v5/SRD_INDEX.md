Use this file for deep links when implementing or validating `wod_chargen/games/lotn_v5/` data. Pair with local `RULES_INDEX.md` (PDF ↔ JSON mapping) and optional pocket PDF (gitignored).

## Cursor @Docs (recommended)

Index the SRD in Cursor so you can cite it in chat:

1. Open **Cursor Settings → Features → Docs**
2. Click **Add new doc**
3. URL: `https://www.oneworldofdarkness.com/laws-of-the-night/`
4. Name: `LoTN V5 SRD` (or similar)
5. Wait for indexing to finish, then use `@LoTN V5 SRD` in prompts

See `reference/lotn_v5/CURSOR_DOCS.md` for details and troubleshooting.

## Maintenance

- SRD is a living site; slugs may shift. Re-verify links when adding data.
- This index is **hub-local reference** — not a scrape of SRD text (respect OWOD/Dark Pack terms).
- When JSON and SRD disagree, note the discrepancy in PR/commit and prefer SRD for MET LARP rules unless Storyteller house-rules apply.
