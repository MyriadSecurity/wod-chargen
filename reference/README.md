### Cursor @Docs (quick start)

1. **Settings → Features → Docs → Add new doc**
2. URL: `https://www.oneworldofdarkness.com/laws-of-the-night/`
3. Name: `LoTN V5 SRD`
4. In chat: `@LoTN V5 SRD` + your question

### Local PDF (optional)

PDF files are **gitignored** — keep your pocket-book copy only on your machine:
